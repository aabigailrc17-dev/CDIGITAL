3. Manual de Usuario e Informe Técnico Básicos

Este apartado documenta la infraestructura de desarrollo de software, la planeación y la gestión del tiempo invertido en la construcción del sitio web[cite: 40].

Herramientas Empleadas en el Desarrollo

  • Editor de Código: Visual Studio Code (v1.90), utilizado para la codificación estructurada del lenguaje de marcado y hojas de estilo.
  • Herramientas de Productividad: Emmet (integrado en VS Code), empleado para la generación acelerada de layouts HTML por medio de atajos de teclado.
  • Entorno de Pruebas: Extensión Live Server (desarrollada por Ritwick Dey), implementada para la renderización automática en tiempo real de los cambios en el navegador local.
